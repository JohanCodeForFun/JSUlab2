Kunder om resan

Recension:

Hur det fungerar:

  1. Du väljer en av våra tre fiskeorter
  2. Välj hur många nätter du vill fiska
  3. Välj hur du vill bo
  4. Beräkna priset på resan hur du reser
  5. Klicka på ‘Räkna på priset’ för att få beräknad kostnad

Använd vår kalkylator här för att få ett beräknat pris på resan!
